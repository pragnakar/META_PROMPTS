# Scrum

This repository holds the **Scrum** meta-prompt: agile planning, sprint cycles, backlog management, roles, ceremonies, and team coordination practices.

## Structure
- `AGENT.md` - Agent directives for Scrum facilitation and sprint execution
- `SPEC.md` - Scrum process specification, definitions of done, and cadence guidelines
- `BUILD_LOG.md` - Sprint logs, retrospective findings, and process improvements

## Usage
Use this meta-prompt when your project follows Scrum or agile iterative planning.
