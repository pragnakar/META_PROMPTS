# AGENTS.md

This project uses the **Meta-Prompts** protocol for AI-assisted software development.

## For AI Agents

**If the project is not yet initialized (no `.build/` directory):**

Read and follow the bootstrap protocol at:
```
meta_prompts/BOOTSTRAP.md
```

Do not write code, create application files, or make architectural decisions until the initialization sequence in `BOOTSTRAP.md` is complete and the human has approved the Phase 1 specification.

**If the project is already initialized (`.build/` directory exists):**

Read these files in order:
1. `.build/AGENT.md` — your directive, constraints, and tech stack
2. `.build/SPEC.md` — current phase specification and acceptance criteria
3. `.build/BUILD_LOG.md` — last entry shows exactly where the build left off

Report to the human what you found and resume from the last log entry.

## Meta-Prompts location

`meta_prompts/` — contains 11 meta-prompts covering the full software development lifecycle.
